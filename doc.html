Tester.js

tester.js creates an instance of the miniExpress server
and starts listening on port 9134(to avoid interference with the system)
Every 1 second a request is being made to the server using the http module.
The in order they appear are:

Static tests:

1. Get the index.html from the server explicitly. -(1924 bytes)
2. Get the index.html by only mentioning the directory path using the conventional behaviour. - (1924 bytes)
3. Try to get a resource not present on server. - status 404
4. Send a POST request to the server with no POST handler. - status 404
5. A request with an illegal method - parsing error. - status 500
6. Try and initiate a traversing attack on the server. - status 404
7. Get the index.js javascript file. - (763 bytes)
8. Get the style.css CSS file. - (2787 bytes)
9. Get the photo.jpg image file. - (62915 bytes)
10. Get the profile.html file and test the 'Connection:close' header. - (1924 bytes)
11. Test unsupported HTTP/2.1 protocol. - Expect status 500 then a time-out in 2 seconds
12. Test sending multiple requests on same connection (concatenated requests). - Expect both files.
13. Test sending a request where the body is separated from the headers (chunked) - Expect the file asked. (2073 bytes) and time-out (179 bytes)

Dynamic tests:

1. Create a server with POST and GET handlers with bodyParser and CookieParser.
2. Print the app.routes .
3.
  1. Send POST request to the parametrized (:name) path with urlencoded body and cookies.
  2. Print the params (Expect: sshenz62 as :name).
  3. Print urlencoded decoded body.
  4. Print Cookies.
  5. Send status 200.(Expect: 24 bytes of body).
4.
  1. Send GET request with url query ?q=house+horse.
  2. Print decoded query.
  3. Set 2 set-cookies to the client with properties.
  4. Send status 200. (Expect 24 bytes of body and the cookies).

Remarks:

To be able to test all file formats the test only print the length of the body (Except for tests 10,12)
without the content.

After the tests the server waits for 1 second for a time-out and closes.
Overall tester duration is 23 Sec.
